### 使用

- fork后仓库，同步源仓库代码

	- ​添加原始仓库为上游远程源​

		- git remote add upstream https://github.com/original_owner/original_repository.git

	- 获取源仓库的所有分支和提交

		- git fetch upstream

	- 切换到本地主分支（假设主分支名为main）

		- git checkout main

	- 将源仓库的主分支合并到本地主分支

		- git merge upstream/main

	- 将更新后的本地主分支推送到自己的远程仓库（即fork的仓库）

		- git push origin main
